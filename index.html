Конвертер тестов Word → JSON

Загрузите Word-файл с тестами по шаблону и получите JSON для вашего приложения

Загрузите Word-файл с тестами

Перетащите файл сюда или нажмите для выбора

Поддерживаемые форматы: .docx, .doc, .txt

Требуемый формат файла:

Вопрос 1?
правильный вариант
неправильный вариант
неправильный вариант
неправильный вариант
неправильный вариант

Вопрос 2?
правильный вариант
неправильный вариант
неправильный вариант
неправильный вариант
неправильный вариант

Важно:

  • Каждый вопрос должен заканчиваться знаком вопроса "?"
  • Первым вариантом после вопроса должен быть правильный ответ
  • После вопроса должно быть ровно 5 вариантов ответов (1 правильный + 4 неправильных)
  • Между вопросами должна быть пустая строка
